# Einmalige Dataset-Importer

Lege hier nur kleine, datasetspezifische Skripte ab, die Originalannotation → Standardmanifest überführen.

Beispiele:

```text
build_watkins_manifest.py
build_cbi_manifest.py
build_dcase_manifest.py
```

Die eigentliche Benchmark-Pipeline soll nicht wissen, wie ein spezielles Dataset intern strukturiert ist.
